# 🏥 Healthcare Dashboard

A responsive healthcare dashboard built using HTML, CSS, and JavaScript that displays patient data, vital signs, and diagnosis history using an external API.

---

## 🚀 Features

- 📊 Blood Pressure Chart (Chart.js)
- 👤 Patient Profile (Jessica Taylor)
- ❤️ Vital Signs Display (Respiratory Rate, Temperature, Heart Rate)
- 📋 Diagnostic List Table
- 🎨 Clean and modern UI layout
- 🔐 API integration with Basic Authentication

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Chart.js
